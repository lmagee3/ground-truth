◉ WHAT IS GROUND TRUTH?
An open-source geopolitical context engine that answers one question:

"How did we get here?"

World Monitor shows you WHAT is happening on the globe right now.
Ground Truth shows you WHY it's happening — the 10, 20, 50-year chain of events, treaties, betrayals, economic pressures, and power shifts that led to this exact moment.

No media narrative. No spin. No feelings spared.
Just primary sources, verified data, and structured historical context — delivered as an API that anyone can integrate.
◎ CORE PRINCIPLES

🔒 Source Transparency

Every claim links to its primary source. Government archives, declassified documents, treaty texts, economic data — never secondary media. Users can verify everything themselves.

⚖️ Engineered Neutrality

Not "we promise to be unbiased." Instead: every context report presents multiple competing narratives with the evidence for each. The engine doesn't pick sides — it shows the data and lets the reader decide.

📡 Primary Sources Only

No Wikipedia (editable by anyone). No news articles (editorial spin). Library of Congress, National Archives, CIA Factbook, UN resolutions, declassified intel, treaty databases, economic indicators from central banks. The real deal.

🔧 Tool, Not Publication

Ground Truth is an API and engine first, not a blog or media outlet. Other platforms (World Monitor, newsrooms, researchers) integrate it. We build infrastructure, not content treadmills.

⚙ HOW A QUERY WORKS
1. INPUT → User queries: "Why are there tensions in the South China Sea?"
2. GDELT SCAN → Pull all events in SCS region, last 30 days + historical spikes
3. ACLED/UCDP → Overlay conflict events, military movements, diplomatic incidents
4. HISTORICAL PULL → NARA: declassified cables on SCS. LOC: treaty texts (UNCLOS). CRS: congressional reports on China-Pacific policy
5. ECONOMIC LAYER → World Bank: trade dependencies. UN Comtrade: shipping volumes. USITC: tariff data
6. ARMS CONTEXT → SIPRI: arms transfers to regional actors. Military spending trends
7. AI SYNTHESIS → Claude/local model generates structured timeline with multiple interpretive frameworks
8. QA PASS → Antigravity verifies dates, sources, checks for bias
9. OUTPUT → JSON (API consumers) + Markdown (human readers) + embeddable HTML
